kick-off, git, and summer break'21

Aug 1 2021

Hello!

insert drumroll, perhaps some applause
(Just kidding.)

I am so, so pumped to be writing the first blog for my website. Building something of my own has always been this huge goal of mine and to see thearvindas.me come to life is unreal to me.

The process started way back at the start of June. I started looking up static site generators and, needless to say, I was very flustered by the endless resources and results. I finally found a theme I liked on Hugo and henceforth, began the journey of scrolling through documentation, learning Git and watching hours of YouTube, understanding how to work with Hugo.

It was all worth it, though. 100%. I’m really pleased with how this has turned out, and like I said, writing this first blog is such a nice feeling. But, what do I plan to write here? I initially thought I could perhaps, make it a portfolio but I finally decided to make it a documentation of what I learn, what inspires me, and maybe some F1 now and then.

Today is the 1st of August. Summer break ends in a couple of weeks, and oh boy, what a summer it has been. This blog is, however, just the introduction. Summer break stories for another day.

If you’ve read this far, thank you so much. I hope this website can add some value to you in the days to come.

Peace out.

(PS, huge shoutout to Manan for always inspiring me to build something of my own; this website wouldn’t be here if not for him)

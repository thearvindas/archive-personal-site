my favourite reads, tools, and other random things on the internet [keeps updating]

Dec 17 2021

The following is a non-exhaustive gratitude list [keeps updating] of all my favourite reads, websites, tools, and any other random thing I’ve found on the internet that has helped me make my life better.

reads#

  1. The Theory of Cumulative Stress: How to Recover When Stress Builds — James Clear → great read about how stress builds up and how you can fight back

  2. Landing at Razorpay — Manan Agarwal → fantastic read about hard work, humility, and trusting the process

  3. Rules — Aditya Mohanty → this read is what inspired me to define rules for my own life

  4. From Fool To Cool: How To Stay Calm, No Matter What — Bruce Kasanoff → fascinating read about staying calm

  5. Lines, not dots — Manas Saloi → a great post (as always from Manas) about building professional relationships

websites#

  1. MonkeyType → hands down the best website to practice touch-typing; I’ve been using it for nearly a year, can average around 85-90wpm now
  2. Laws of UX → brilliantly designed website with principles of design that you should keep in mind

tools#

  1. Markdown New Tab → love using this for my new tabs, easy to jot down to-dos, reminders, etc
  2. Visualize Value → when I’m not using Markdown New Tab, Visualize Value is my go to extension for new tabs. Huge fan of the minimal illustrations, massive value add with every new tab
  3. Spotistats → the dopamine hit that Spotify Wrapped gives, but any time of the year
